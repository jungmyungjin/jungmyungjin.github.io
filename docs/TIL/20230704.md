### **✔️ JSDoc 사용법**

사용법은 간단합니다. 아래와 같이 입력하면 됩니다.
```jsx
/**
 * 이름과 나이 입력받아 출력하는 함수
 * @param {string} name 이름 입력란 입니다!
 * @param {number} age 나이 입력란 입니다!
 * @returns 이름과 나이를 출력합니다~
 */
const lime = (name, age) => {
  return name + age;
};
```
